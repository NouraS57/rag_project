# RAG Project

## 📌 Description
Ce projet est un système de Retrieval-Augmented Generation (RAG) permettant de répondre à des questions en se basant sur des documents.

## ⚙️ Technologies utilisées
- Python
- LangChain, streamlit , chromadb

## 📁 Structure du projet
- src/ → code principal
- main/ → application
## 🚀 Comment exécuter le projet
1. Lancer le programme :
   python app.py
